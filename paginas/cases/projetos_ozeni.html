Ozeni Lima

Escritora com livros publicados tanto no Brasil como Portugal - 2017



O projeto

Escritora que também trabalha na FATEC-RL, ela pediu ajuda de uma equipe de alunos para que pudessemos realizar a criação de um site e blog para que ela escrevesse seu conteúdo.

A solução

Escolhemos wordpress pelo pedido de um gerenciamente simples de conteúdo. A escritora dizia ter dificuldade para se achar na internet e queria algo bem simples que pudesse ajudar na hora de passar suas ideias. Escolhemos então o wordpress e fomos fazendo mockups até a mesma autorizar a próxima etapa.

Minhas funções

A experiência foi extremamente rica, tanto para aprender a ter um contato com os clientes quanto para modificar o wordpress ao máximo. Fiquei encarregado de passar a ideia do cliente para o wordpress, fazendo a instalação e modificação do futuro tema e alterações no css e js para criação da home page. Os meus colegas de projeto acabaram ficando com o gerenciamento de conteúdo, contato com o cliente, promoção do canal do youtube e instalação da área de contato.
