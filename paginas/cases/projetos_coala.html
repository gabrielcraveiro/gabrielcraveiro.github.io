Conceito de doceria - 2017



O projeto


Foi solicitado uma criação de um mascote e identidade visual para uma possível doceria com o nome de Coala Doces. Realizei então todo o processo desde desenho do logo até implementação de mockups. Usei também cores pasteis que promovem a calma para o cliente e combinam com o tom da doceria.
